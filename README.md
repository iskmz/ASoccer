# ASoccer

ASoccer Application is an iOS app which presents an up-to-date soccer data (competition standings) for clubs worldwide. 
The App uses backendless database (backend) with codeless-built server-side APIs.
It has user registration with login and a responsive UI for all iPhone devices.

something important!!!
----------------------
1- if you cloning the project and it doesn't work, and in the target debugger(in XCode) says something about not recognizing             Backendless:
just open terminal window and go the project file and enter 'pod update'

2- if you getting another problem in the target debugger that you don't have permission for a file that in the target debugger you need to open terminal window and enter the project folder and keep digging (cd filename) within the location of this file that you will see in the target debugger and change the permission of this specific file by entering (chmod 777 filename)
